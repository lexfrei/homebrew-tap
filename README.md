# homebrew-tap

Homebrew tap for [lexfrei](https://github.com/lexfrei) projects.

## Usage

```bash
brew tap lexfrei/tap
```

## Available Formulae

| Formula | Description |
| --- | --- |
| [claudeline](https://github.com/lexfrei/claudeline) | Claude Code statusline with real usage limits from Anthropic API |

### Install claudeline

```bash
brew install lexfrei/tap/claudeline
```
